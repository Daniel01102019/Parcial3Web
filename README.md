# Apple Store 3D - React Three Fiber

Proyecto inspirado en la página oficial de Apple, desarrollado con React y React Three Fiber.

## Características

- Visualización 3D interactiva de modelos iPhone
- Cambio dinámico de colores
- Selección de capacidad y cantidad
- Carrito de compras dinámico
- Responsive para dispositivos móviles
- Loader personalizado con Suspense
- Efecto de paralaje con el mouse

## Tecnologías utilizadas

- React
- React Router DOM
- React Three Fiber
- Drei
- Tailwind CSS
- Vite

## Librerías adicionales

- @react-three/fiber
- @react-three/drei

## Instalación

```bash
npm install
npm run dev
```
